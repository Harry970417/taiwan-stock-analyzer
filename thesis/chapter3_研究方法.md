# 第三章　研究方法

## 3.1　研究設計總覽（V1探索性架構）

本研究採先導性探索設計（V1 Pilot Study），以16檔台灣上市股票為研究標的，針對六項量化選股因子在485個交易日觀測期間（2024年6月12日至2026年6月12日）之截面預測能力進行系統性實證。V1設計定位為探索性（exploratory）研究，目的在於建立可重現之量化因子評估框架，識別值得後續大樣本驗證研究（V2）進一步檢驗之研究方向。

### V1設計之定位原則

V1設計在統計推論上具有以下三項核心原則：

（一）保守顯著水準設定。H1採雙尾α=0.10，相較於慣用α=0.05較為寬鬆，此選擇反映在J=6之小樣本條件下，精確排列檢定之統計功效（statistical power）受限，過度保守之顯著水準可能導致真實方向性信號無法被偵測。

（二）推論措辭之謹慎界定。所有統計推論均限定於「本研究觀測樣本中之初步方向性觀察」，不宜直接推廣至全市場或其他觀測期間。樣本規模之侷限使各項統計量具有較高估計不確定性，研究結論須審慎詮釋。

（三）三項假說推論相互獨立。H1、H2、H3各自採用最適合其研究問題之統計方法，推論結果獨立解讀，不進行跨假說之聯合推論。三項假說共用相同股票池與因子資料，但對應不同分析層次：排名一致性（H1）、條件IC差異（H2）、及Alpha顯著性（H3）。

### 三項研究假說與對應方法

| 假說 | 核心問題 | 統計方法 | 顯著水準 |
|------|---------|---------|---------|
| H1 | 六因子IC均值排名與Sharpe比率排名是否具單調一致性？ | Spearman精確排列檢定（J=6，720排列） | 雙尾 α=0.10 |
| H2 | 財報公告事件期IC是否系統性低於非事件期IC？ | Newey-West HAC單尾t檢定 | 單尾 α=0.05 |
| H3 | EPS年增率因子於高IC季度之多空組合是否存在顯著正超額報酬？ | Jensen Alpha單一指數模型＋Newey-West HAC | 單尾 α=0.05 |

三項假說在概念上具有遞進關係：H1確認因子截面預測能力之排名結構是否與多空策略績效排名一致；H2探討財報公告事件對因子效能之干擾機制，作為IC-Portfolio Divergence的替代解釋；H3在控制系統性市場風險後，直接以Jensen Alpha量化高IC季度多空投資策略之超額報酬。

### 資料期間

本研究觀測期間為2024年6月12日至2026年6月12日，共485個有效交易日。期間涵蓋台灣股票市場之不同短期景氣階段，為六項因子在實際市場環境下之截面預測能力提供初步觀察基礎。

### 股票池建構程序

本研究以台灣證券交易所上市普通股為候選標的，並依以下三道篩選程序建立有效股票池：

（一）資料完整性篩選。研究期間內須有至少100個有效交易日，排除資料中斷、上市未滿一年，或觀測期間內有重大停牌紀錄之個股，確保截面IC計算具備基本資料基礎。

（二）流動性篩選。以個股日均成交量作為流動性代理指標，排除流動性嚴重不足之標的，以確保因子信號在實際投資情境下具執行可行性，避免理論回測績效因流動性障礙而難以落實。

（三）財務資料可得性篩選。以FinMind API作為主要財務資料來源，篩選條件要求各個股之財務資料覆蓋完整，確保因子面板資料可被完整建構。

### 股票池規模限制說明

16檔之股票池規模為本V1探索性研究的設計限制，此規模低於截面因子分析之學術建議標準（通常建議50檔以上）。規模不足導致每日有效截面觀測數偏低，截面IC估計具有較高抽樣變異。此限制在後續V2研究中將透過擴展至全市場範圍加以改善。本研究之所有統計推論均在此樣本侷限下進行，並於§4.9完整揭露相關研究限制。

### 資料來源

| 資料類型 | 資料來源 | 頻率 |
|---------|---------|------|
| 個股日收盤價、成交量 | FinMind API | 日頻 |
| 季度每股盈餘（EPS） | FinMind API | 季頻 |
| 月營收 | FinMind API | 月頻 |
| 市場指數（TWII） | FinMind API | 日頻 |

所有資料均以FinMind API完整版取得，以確保EPS年增率與月營收年增率因子之財報季度覆蓋完整性。市場代理指數採台灣加權股價指數（TWII），作為H3假說Jensen Alpha估計之系統性風險基準。

## 3.3　六因子定義與公告延遲設定

本研究共計算六項量化選股因子，涵蓋基本面動量、技術趨勢及量能等不同資訊維度。各因子之定義、計算頻率及公告延遲設定如下：

| 因子名稱 | 英文代號 | 計算公式 | 頻率 | 公告延遲 |
|---------|---------|---------|------|---------|
| 月營收年增率 | Revenue YoY | $(Rev_t - Rev_{t-12}) / \|Rev_{t-12}\|$ | 月頻 | +10個交易日 |
| EPS年增率 | EPS Growth | $(EPS_t - EPS_{t-4}) / \|EPS_{t-4}\|$ | 季頻 | +45個交易日 |
| 20日動能 | Momentum20D | $P_t / P_{t-20} - 1$ | 日頻 | 無 |
| 成交量比 | Volume Ratio | $V_t / \overline{V}_{20d}$ | 日頻 | 無 |
| RSI-14 | RSI-14 | Wilder RSI，14日回顧期 | 日頻 | 無 |
| MACD訊號 | MACD Signal | MACD信號線（9日EMA） | 日頻 | 無 |

### 公告延遲設計原則

EPS年增率與月營收年增率屬財報揭露型因子，須設定公告延遲以防止前瞻偏誤（Look-ahead Bias）。本研究採保守設定：月營收公告延遲+10個交易日，對應上市公司依法規於次月10日前公告之時間節點；EPS季報公告延遲+45個交易日，模擬投資人於財報合法公開後方能取得資訊之現實限制，避免研究回測納入尚未公開之盈餘資訊。技術面因子（動能、成交量比、RSI-14、MACD訊號）基於當日市場可觀察資料計算，不涉及公告延遲。

### 截面資訊係數（IC）定義

各因子截面IC定義為：在每一有效交易日$t$，計算全股票池之因子值排名與次日個股報酬排名之Spearman等級相關係數：

$$IC_t = \rho_S\left[\text{rank}(F_{i,t}),\ \text{rank}(r_{i,t+1})\right], \quad i \in \text{Universe}_t$$

其中$F_{i,t}$為股票$i$在$t$日之因子值，$r_{i,t+1}$為$t+1$日個股報酬，$\mathrm{Universe}_t$為當日有效股票集合，且每日截面至少需5檔有效觀測方進行計算，否則跳過該日。跨日匯總統計量定義如下：

$$\overline{IC} = \frac{1}{T}\sum_{t=1}^{T}IC_t, \qquad ICIR = \frac{\overline{IC}}{\sigma(IC)}, \qquad t\text{-stat} = ICIR \times \sqrt{T}$$

## 3.4　研究假說設計

本研究三項假說的設計源自一個實證觀察：量化選股因子之截面資訊係數（IC）統計量，與該因子所建構之多空投資組合實際績效之間，存在系統性背離，即所謂的IC-Portfolio Divergence現象。此背離的成因可從排名結構一致性、財報事件干擾，及策略執行障礙三個層次加以探討，分別對應本研究三項研究假說。

假說H1旨在確認IC排名結構與組合績效排名之間的基本關係。若六項因子之截面IC均值由高至低所形成的排名，能與其對應多空組合夏普比率的排名呈現一致性，則表示IC均值作為因子有效性衡量工具，在預測策略相對績效上具有結構性支持。反之，若兩者排名缺乏一致性，IC均值的代理效度即存在根本疑問。H1在本研究中定位為前置性方向性驗證，其結論為後續H2、H3提供邏輯基礎，而非決定性前提條件。

假說H2從市場微觀結構角度提出解釋。財報公告屬於高資訊密度事件，在公告前後一定交易日窗口內，個股報酬率主要由盈餘訊息及市場即時反應所主導，截面因子信號的相對預測力可能遭到壓縮，導致事件期IC系統性低於非事件期IC。若H2成立，IC均值的計算納入事件期觀測值，將低估因子在常態截面環境下的預測能力，從而部分解釋IC數值偏低但組合績效相對優異之背離現象。

假說H3從策略執行障礙角度提供另一解釋路徑。台灣股票市場之放空機制存在借券限制與執行成本，使因子排名最低之股票難以被充分放空套利。若短端執行障礙確實存在，即使因子截面IC在特定季度偏低，多空組合之空方部位仍無法被市場有效定價，導致多空組合報酬未如IC所預期呈現負值。H3以EPS年增率因子為焦點，分別在高IC季度（Q5）驗證多空組合是否存在顯著正超額報酬，並在低IC季度（Q1）確認超額報酬是否如無障礙情境之預期為負值。若Q1組合之Alpha未呈現顯著負值，甚至維持非負方向，則在本研究觀測樣本中與短端執行障礙假說所預期之方向一致。

三項假說的邏輯依存關係如下：H1提供因子效能的整體排名視角，H2與H3則分別從事件干擾與執行障礙兩個機制，提供IC-Portfolio Divergence之可能解釋，三者共同構成本研究的假說推論體系。

## 3.5　研究變數與衡量架構

本研究在三項假說中分別運用不同的核心變數，各變數在推論架構中扮演明確且不可互換的角色。以下說明各變數之定位，並指明其對應之假說用途。

**因子值（Factor Value）**
為各因子之截面觀測量，代表個股在特定日期之原始信號強度。本研究六項因子值的計算依§3.3所定義之公式與公告延遲設定產生，以日頻面板形式呈現。因子值本身是截面排序的依據，不直接作為推論變數，而是後續IC計算與組合建構之輸入來源。

**截面資訊係數（IC）**
為每日因子值排名與次日個股報酬排名之Spearman等級相關係數，衡量因子在當日截面對次日相對報酬之排序預測能力。IC在三項假說中的角色各有不同：在H1中，以全研究期間之IC均值代表各因子整體截面預測能力，並與Sharpe比率排名進行對比；在H2中，IC依財報事件期與非事件期分別彙總，以觀察事件干擾對因子預測能力的條件性影響；在H3中，IC以季度為單位彙總，作為區分高IC季度（Q5）與低IC季度（Q1）的分類依據。

**多空組合報酬（Long-Short Portfolio Return）**
為每日因子五分位最高組與最低組等權平均報酬之差值，代表純截面多空策略在當日之報酬實現。

**夏普比率（Sharpe Ratio）**
以全研究期間多空組合報酬之年化超額報酬與年化波動度之比值計算，作為H1中衡量各因子策略長期績效之標準化指標，與IC均值排名進行等級對比。

| 假說 | 自變量 | 因變量 | 推論方向 |
|------|-------|-------|---------|
| H1 | 各因子IC均值 | 各因子Sharpe比率 | 兩排名之等級一致性 |
| H2 | 事件期/非事件期分類 | 季度IC差異（$d_q$） | 非事件期IC是否系統高於事件期 |
| H3 | IC季度高低分類（Q5/Q1） | 多空組合Jensen Alpha | Q5 Alpha是否顯著為正；Q1 Alpha是否未呈現顯著負值 |

## 3.6　研究流程

本研究依以下八個步驟依序執行，形成一套可完整重現之量化研究流程，各步驟之技術細節詳見第四章。

**步驟一：資料取得。**
以FinMind API為主要資料來源，取得研究樣本股票於觀測期間（2024年6月12日至2026年6月12日）之日頻股價、成交量，以及季度EPS與月營收等財務資料；市場指數資料以台灣加權股價指數（TWII）為代理，同步取得。所有財務資料依§3.3所定義之公告延遲規則處理，確保資料取用不含前瞻偏誤。

**步驟二：股票池建構與確認。**
依資料完整性、流動性及財務資料可得性三道篩選標準，於候選標的中確認最終有效股票池，並記錄篩選過程與排除原因，確保股票池組成之可追溯性與透明度。

**步驟三：六因子建構。**
依§3.3所定義之因子公式與公告延遲規則，以股票池16檔個股為截面，計算六項量化選股因子之每日因子值：月營收年增率（Revenue YoY，+10個交易日延遲）、EPS年增率（EPS Growth，+45個交易日延遲）、20日動能（Momentum20D）、成交量比（Volume Ratio）、RSI-14，以及MACD訊號（MACD Signal）。各因子依其頻率與延遲設定逐日更新，形成研究期間之因子面板資料，作為步驟四截面IC計算之輸入來源。

**步驟四：截面IC計算。**
依每一有效交易日之橫截面，計算各因子值排名（Factor Rank）與次日個股報酬排名（Next-Day Return Rank）之Spearman等級相關係數，得每日截面IC值。跨日匯總統計量（IC均值、標準差、ICIR、t統計量）依§3.3所定義之公式計算，形成各因子之IC描述統計，為H1、H2、H3三項假說推論提供核心資料基礎，技術細節詳見§4.3至§4.4。

**步驟五：五分位多空組合建構。**
依每日各因子值將股票池均分為五組，建構各因子之Q5多空組合日報酬序列，計算全研究期間年化夏普比率供H1使用；H3另依EPS年增率因子之IC季度分類，分別建構Q5季度與Q1季度之多空組合日報酬序列，供Jensen Alpha估計使用。

**步驟六：三項假說統計推論。**
依第四章§4.6至§4.8之操作規格，依序執行H1精確排列檢定、H2 Newey-West HAC均值t檢定，以及H3 Jensen Alpha單一指數模型估計，輸出各假說之統計量與推論結論。

**步驟七：穩健性分析。**
針對H3 Q5多空組合，在三種無風險利率情境（rf=0%、rf=1.5%、rf=3%）下重新估計Jensen Alpha及對應t統計量，確認主要結論不因利率假設之選擇而實質改變。

**步驟八：實證結果輸出。**
將H1、H2、H3統計推論結果及穩健性分析結果彙整，以第五章表格格式呈現，並依第六章架構進行結論、研究限制、理論與實務意涵之系統性討論。

## 3.7　V1探索性研究限制與V2延伸方向

本研究於V1探索性設計階段即預見若干統計推論之結構性侷限，並於設計階段明確界定研究結論之有效範圍，以確保推論過程之學術誠實性。

**樣本規模侷限。**
本研究股票池16檔，遠低於截面因子分析之學術建議標準（通常建議50檔以上）。此規模導致每日有效截面觀測數偏少，IC估計具有較高抽樣不確定性；多空組合每組平均僅3至4檔，個股特異風險無法有效分散，組合績效易受個別重大事件干擾。485個交易日（約兩年）之觀測期間較短，且集中於特定市場環境，所有統計推論結論均僅代表本研究觀測樣本中之初步方向性觀察，不宜直接推廣至其他市場環境或時間段落。

**H2有效季度不足之風險。**
H2推論有賴足夠數量之有效財報季度，以確保Newey-West HAC估計之漸近可靠性，一般建議有效季度數Q不少於4個。本研究期間內有效財報季度數受資料期間之結構性侷限，若有效季度數未達此門檻，H2之統計推論將因有效季度數不足而無法執行或無法獲得可靠估計結果，H2假說亦無法在本研究條件下進行評估。此為H2推論之關鍵結構性風險，已於第四章§4.9完整揭露。

**多重比較問題。**
本研究在同一資料集上執行四項統計推論程序，構成多重比較情境。個別p值在多重比較條件下，研究整體之型一錯誤率將高於各推論所宣稱之名義顯著水準。本研究於探索性定位下揭露此問題，正文推論不施用多重比較校正，並建議讀者在綜合解讀各假說結論時，將此因素納入整體評估。

**V2延伸研究方向。**
本研究V1設計之核心貢獻在於建立可重現之量化因子評估框架，並在有限樣本條件下識別值得後續深入驗證之研究方向。V2後續研究應在以下面向予以系統性擴展：第一，股票池擴大至涵蓋台灣上市上櫃全市場範圍，建議50檔以上，以確保截面IC估計具備充分統計基礎；第二，觀測期間延長至涵蓋完整市場週期，以降低特定行情環境對研究結論之影響；第三，有效財報季度數擴充至Q≥8以上，使H2 Newey-West HAC推論具備可靠之漸近基礎；第四，因子數量適度擴展，以提升H1排名分析之覆蓋廣度與統計功效；第五，導入樣本外驗證（walk-forward validation）框架，以驗證因子策略在非訓練期間之可複製性。上述擴展均以本V1探索性結果所識別之方向為出發點，V2研究將採確認性（confirmatory）設計取代本研究之探索性定位，以更嚴格之統計標準進行驗證。
